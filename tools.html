Tools:

These are a few tools that I like to use from time to time, some are my own, some have been created by others.

Software:

  • PID Control Library A PID Control Library written in C++
  • Regular Expression Message Validator Great for deployment on embedded systems that require context passing and mode switching features.
  • sshuttle A handy proxy/VPN access to geo-fenced data tool. Especially useful when you need access to resources that are local only. ;)
